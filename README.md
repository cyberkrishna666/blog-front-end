<h1>Blog app</h1>
<br>
	<p><b>Blog application written using React, Express.js and MongoDB.</b></p>
	<p><em>Main functionality:</em></p>
<ul>
	<li>
		Registration
		<ul>
			<li>
				Upload avatar
			</li>
		</ul>
	</li>
	<li>
		Authentication using JWT
	</li>
	<li>
		Roles: Admin, User, Banned
	</li>
	<li>
		Add, edit, delete posts
		<ul>
			<li>
				Upload image
			</li>
			<li>
				Add tags
			</li>
		</ul>
	</li>
	<li>
		Like/Unlike post
	</li>
	<li>
		Add/delete comments
	</li>
	<li>
		Pagination
	</li>
	<li>
		Carousel
	</li>
	<li>
		Modal window
	</li>
	<li>
		"Scroll-to-top" component
	</li>
	<li>
		Search bar
	</li>
	<li>
		Responsive design
	</li>
</ul>
